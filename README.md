* #### Digital Lost \& Found Tracker



A simple Java Swing application that helps track lost and found items.



* **Features:**



  -Add lost or found items

  -Date tracking for each item

  -Automatic match alert when a lost and found item share the same name and location

  -Search items by name or location

  -Delete items

  -Local file storage using a text file



* **Technology:**



Java

Swing (GUI)

File Handling
